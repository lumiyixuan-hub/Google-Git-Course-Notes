# Google: Introduction to Git and GitHub — Course Notes

This repository contains my personal notes and practical implementations from the [Introduction to Git and GitHub](https://www.coursera.org/learn/introduction-git-github) course offered by Google on Coursera.

## 🎯 Course Objectives
The goal of this course is to understand how version control is used in a professional software development environment. Key learning outcomes include:
* Understanding the logic behind Version Control Systems (VCS).
* Mastering local Git operations (staging, committing, branching).
* Collaborating with others using GitHub (remotes, pull requests, merging).
* Troubleshooting and undoing common mistakes in Git.
